## 📌 1. Краткая информация

Training Planner — программа для персонального компьютера, помогающая вести учёт тренировок. Пользователь заполняет короткую форму (дата, вид спорта, продолжительность), а приложение сохраняет сведения в удобную таблицу и автоматически записывает их в JSON-файл.

Код написан на Python с применением Tkinter. Все нужные модули уже есть внутри самого Python, поэтому ничего дополнительно устанавливать не придётся.

---

Скопируйте проект к себе на компьютер командой:

```bash
